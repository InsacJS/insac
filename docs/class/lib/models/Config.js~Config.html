    • optional

    Carpeta donde se encuentran los modelos.
    config.path.routes String
    • optional

    Carpeta donde se administran todas las rutas.
    config.path.seeders String
    • optional

    Carpeta donde se encuentran los seeders.

    • optional

    • optional

    • optional

    Objeto que contiene los campos de configuración para el sistema de autenticación.

    • optional

    Se actualiza la configuración solamente para el modo de ejecución establecido al momento de crear el objeto. Es posible agregar datos adicionales al archivo de configuración, diferentes a los predefinidos.
